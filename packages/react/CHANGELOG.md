# @tailgrid/react

## 0.1.0

### Minor Changes

- Initial release of TailGrid - AI-native data grid for React

  Features:
  - Core grid functionality with sorting, filtering, pagination, and row selection
  - Virtual scrolling for 100k+ rows
  - AI query parsing with OpenAI, Anthropic, Ollama, and custom providers
  - React components and hooks
  - Three styling options: Plain CSS, Tailwind CSS, CSS-in-JS
  - Full TypeScript support
  - WCAG 2.1 accessibility compliance

### Patch Changes

- Updated dependencies
  - @tailgrid/core@0.1.0
